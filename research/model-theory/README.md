# Model Theory

This folder contains the current mathematical and project-contract source for
Voxter.

`voxter-v1.tex` and the rendered `voxter-v1.pdf` define:

- the visuomotor-agent thesis
- observation and binary held-state action representation
- causal alignment rules for datasets
- capture and dataset metadata expectations
- Stage 1 reactive behavioral cloning
- Stage 2 memory-based sequential policy
- Stage 3 reinforcement fine-tuning
- offline and online evaluation protocols
- real-time deployment constraints
- privileged-information restrictions
- known failure modes

When implementation details conflict with this theory document, update the
theory or document the implementation-specific deviation explicitly.
